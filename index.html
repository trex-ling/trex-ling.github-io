hexo 添加图片

发表于 2019-10-15

hexo在博客中插入图片

1.先去blog根目录的_config.yml中把post_asset_folder 的false 改为true，这样是为了使你在新建博客时会同时产生一个同名的文件夹，把与该博客有关的所有要插入的文件放在里面（包括图片），保存退出

2、然后打开终端，在博客的根目录下执行 npm install https://github.com/CodeFalling/hexo-asset-image –save 命令来进行插件的安装。

3、然后创建一博客 hexo new “picture” 然后查看博客的 ../source/_posts 目录下的文件，会看到存在一个picture文件夹 和 picture.md 文件

4.然后把所有图片放到这个picture文件夹里

5.在写博客的时候，在要插入图片的地方用这个格式

6.hexo g

7.hexo s

8.hexo d推送到网站上

ftp

发表于 2019-10-15

Windows系统的FTP服务器文件共享系统搭建

这个服务器可以把自己电脑的文件夹共享到服务器上，通过IP地址来访问文件夹从而对文件夹进行读写操作

1.打开ftp服务

2.在网络信息服务上添加ftp站点

3.在浏览器网址输入ftp:\192.168.11.1:21上验证测试一下

前面记得要加fpt，因为浏览器默认的事HTTP协议。如果用IE浏览器还可以进入文件管理器中进行图形化操作，这种对于学校实验室的小型局域网内还是很有用处的，这样访问实验室的服务器就不需要用u盘考来考去，而且在寝室或在外面要用服务器上的文件资料时也可以很方便的读取。
